1. Tạo db mysql bằng cách import script trong thư mục script.
    - Create new database với tên là: db_development.
    - import database bằng cách chạy script db trong thư mục database_script [db_development.sql]
2. Mở cửa sổ terminal và cd đến thư mục chứa source code và gõ lệnh : npm install 
=> Để install các module cần thiết.
3. Gõ lệnh: npm run dev 
=> Để chạy ứng dụng.
